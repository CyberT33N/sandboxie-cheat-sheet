# Host-Sync Starship

## Status

This is the secondary / legacy Starship architecture path in this repository.

It is not the preferred default.

## Legacy shape

The host-sync / older Starship shape relied on host-visible binaries such as:

- `C:\Program Files\starship\bin\starship.exe`
- `C:\Program Files\Git\git.exe`

That model is preserved only as a legacy reference while the repository converges on the boxed-owned-toolchain shell contract.
