# Troubleshooting


## SBIE1305 Blockiert, dass sandgeboxtes Abbild geladen wurde - \drive\C\testApp\ffmpeg.dll
- Wenn man die Sandbox nicht löscht, kann es auf jeden Fall passieren, dass noch Altlasten vorhanden sind.

Daher muss man immer sicherstellen, dass man vorher in der Sandbox alle Altlasten komplett löscht. Das heißt:
- Entweder löscht man es von Hand.
- Oder man stellt es so ein, dass die Sandbox, wenn alles geschlossen wird, sowieso von Grund auf alles neu löscht.
